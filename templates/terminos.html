2. Objeto y Aceptación

Los presentes términos y condiciones regulan el uso del software QuickBooks Sales Reporter desarrollado por KH LLOREDA, S.A. para la gestión y análisis de datos de ventas.

El acceso y uso de este software implica la aceptación plena de estos términos y condiciones.

3. Servicios Ofrecidos

QuickBooks Sales Reporter proporciona:

  • 🔗 Integración con QuickBooks Online API
  • 📊 Análisis de datos de ventas mensuales y anuales
  • 📈 Reportes automatizados con cache inteligente
  • 🤖 Integración con OpenWebUI para consultas en lenguaje natural
  • 🔄 Actualizaciones programadas de datos

4. Responsabilidades del Usuario

El usuario se compromete a:

  • Proporcionar credenciales válidas de QuickBooks Online
  • Usar el software conforme a su propósito legítimo
  • No compartir credenciales de acceso con terceros
  • Mantener la confidencialidad de los datos empresariales
  • Cumplir con la normativa de protección de datos vigente

5. Protección de Datos

En cumplimiento del Reglamento General de Protección de Datos (RGPD):

  • Los datos se procesan únicamente para el análisis de ventas
